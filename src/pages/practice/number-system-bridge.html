Supported Input Formats

Adult Mode Supported Formats

  • Supports shorthand like 45.3B (Billion), 10 M (Million), 50k (Thousand).
  • Indian units like 10cr (Crore) and 5L (Lakh) are also supported.
  • You can use aliases: M, Mil, Million all work.
  • Commas are ignored, so 1,000,000 is valid.

Kid Mode Supported Formats

  • Only numbers and a single decimal point are allowed.
  • Decimal values are rounded to two places.
